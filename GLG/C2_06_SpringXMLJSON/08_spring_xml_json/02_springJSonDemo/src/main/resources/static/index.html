• Export JSon ou XML d'un objet
  • Testez plus précisément en ligne de commande : 
                    curl -v -H "Accept: application/json" http://localhost:8080/contact/3
    et 
                    curl -v -H "Accept: application/xml" http://localhost:8080/contact/3
  • Démonstration inverse : POST de valeurs JSON ou XML vers l'application : 
                    curl -d '{"id": 10, "nom": "lovelace", "adresse": "Cambridge", "numeros": [{"typeInfo" : "BUREAU", "numero": "01402654678"}]}' -H "Content-Type: application/json" -X POST http://localhost:8080/contact
    En XML : 
    curl -d '<Contact><id>3</id><nom>nom3</nom><adresse>adresse3</adresse><numeros><numeros><typeInfo>BUREAU</typeInfo><numero>0193378</numero></numeros><numeros><typeInfo>MAISON</typeInfo><numero>06766854</numero></numeros></numeros></Contact>' -H "Content-Type: application/xml" -X POST http://localhost:8080/contact
    Le même sans numéro : 
    curl -d '<Contact><id>3</id><nom>nom3</nom><adresse>adresse3</adresse><numeros></numeros></Contact>' -H "Content-Type: application/xml" -X POST http://localhost:8080/contact
