On peut voir l'image créée par le code.

Sinon :

Récupération de l'image en jpeg : 
        curl -v -H "Accept: image/jpeg" http://localhost:8080/image > toto.jpg
Récupération de l'image en png : 
        curl -v -H "Accept: image/png" http://localhost:8080/image > toto.png
Information sur une personne : 
        curl -v -H "Accept: text/html" http://localhost:8080/personne
Envoi des données d'une personne : 
        curl -d 'Niklaus:Wirth' -H "Content-Type: text/plain" -X POST http://localhost:8080/personne
